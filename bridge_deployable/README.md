Run locally:

1. Install dependencies:

```bash
npm install
```

2. Start server:

```bash
npm start
```

3. Open two browser windows at `http://localhost:3000` to play together.

Controls:

- Left / Right arrows: move piece
- Down arrow: soft drop
- Up arrow or X: hard drop
- Z: rotate

Goal:

Work together to fill the highlighted row (bridge) across the grid.
